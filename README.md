# Festival Pewma Antü — Landing Page

Sitio web del **Festival Nacional e Internacional de Folklore Pewma Antü**, un encuentro de música y danza tradicional nacido en Chile.

## 🌐 Descripción

Landing page moderna, responsive y accesible construida con **HTML + CSS + JavaScript vanilla** (sin frameworks).

### Secciones
- **Hero** con carrusel de fotos autoplay + tarjeta de próxima edición con contador regresivo en vivo.
- **El Encuentro de las Estrellas** — sobre el festival.
- **El Himno Pewma Antü** — reproductor de audio + letra.
- **La visión de Jeremy Armijo** — fundador y director.
- **Nuestro equipo** — integrantes.
- **Mapa interactivo de países participantes** (2025–2027) con tooltips y filtro por año.
- **Galería** de fotos.
- **Noticias, programación e Instagram**.
- **Agrupaciones nacionales** de Chile por comuna.

## 📁 Estructura

```
pewma-antu/
├── index.html
├── css/styles.css
├── js/main.js
├── img/            # Fotografías del festival y logo
└── audio/          # Himno del festival
```

## 🚀 Cómo verlo localmente

Sirve la carpeta con un servidor local (necesario para que carguen los recursos):

```bash
cd pewma-antu
python -m http.server 8080
# Abrir http://localhost:8080
```

También funciona con la extensión "Live Server" de VS Code.
